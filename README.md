# A minimum Demo of LibXaw

X Athena Widgets

## Build

For debian based system
```bash
sudo apt install libxaw7-dev
```
```bash
mkdir build
cd ./build
cmake ..
cmake --build .
./X_Athena_Widgets_Demo
```
